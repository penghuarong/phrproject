bootstrap是基于HTML、CSS、Javascript开发的，在jQuery的基础上进行了更为个性化和人性化的完善，形成一套自己独有的网站风格，并兼容大部分jQuery插件。

优点：简洁灵活、使WEB开发更加快捷。

bootstrap.min.css第三方开源框架/工具包，基于UI（开放源代码）

bootstrap.min.js兼容IE8浏览器
适用于响应式布局多的页面

按钮
使用方法：在元素上添加样式名，前提必须添加公共类
按钮样式名:
.btn （公共类）基础样式（带渐变的标准灰色按钮）
.btn-default  默认按钮
.btn-primary  主要按钮（视觉吸引力强，用于标识一组按钮中最主要的动作）
.btn-success  成功按钮（表示操作成功或动作正确）
.btn-warning  警告按钮（表示动作应该谨慎）
.btn-info  信息按钮（可用于替换默认样式）
.btn-danger  危险按钮（表示危险或有潜在威胁的动作）
.btn-inverse  相反按钮（做为补充的深灰色按钮，与具体行为或动作无关）
.btn-link  链接按钮

按钮的尺寸
.btn-lg  大型按钮
.btn-default 默认按钮大小(中)
.btn-sm  小型按钮
.btn-xs  迷你型按钮

表格
添加在table上的类：
.table   基础表格
.table-striped  斑马线表格[背景交替]

.table-hover  鼠标悬停显示高亮
.table-condensed  紧凑型表格
.table-responsive  响应式表格

添加在tr上的类：
.active  表示当前活动的信息
.success  表示成功或者正确的行为
.info  表示中立的信息或行为
.warning  表示警告，需要特别注意
.danger  表示危险或者错误行为

.nav-list  列表导航
.dropdown 加给li表示下拉样式导航
.breadcrumb  面包屑导航

 第十七单元 bootstrap

	bootstrap 框架(集)
	bootstrap 是一个框架集
	bootstrap 是移动设备优先的
		用于开发响应式布局、移动设备优先的 WEB 项目
	优点: web开发更迅速、简单、更好、更强壮
	bootstrap 的核心是 css样式表
	bootstrap 定义了栅格系统 为 12 列 当大于12时，则换行显示

	超小屏幕（手机，小于 768px） 
	小屏幕（平板，大于等于 768px）
	中等屏幕（桌面显示器，大于等于 992px
	大屏幕（大桌面显示器，大于等于 1200px）

	栅格系统中列  .col-md-*(*代表纯数字，最大值(和)不能超过12)
				  .col-lg-*  超大屏  /  大屏幕  (大桌面显示器 大于等于1200)
				  .col-ms-*  中等屏幕 / 普通屏幕  (桌面显示器  大于等于992)
				  .col-sm-*  小屏幕 (平板  大于等于768)
				  .col-xs-*  超小屏幕  (手机 小于768)

	列偏移 .col-md-offset-*  
		(与上同理)
